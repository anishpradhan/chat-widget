Live Chat Widget Demo

This page demonstrates the embeddable live chat widget. The widget appears as a floating chat button in the bottom-right corner. Click on it to start a conversation!

Features:

  • ✅ Real-time chat with WebSocket support
  • ✅ Responsive design for mobile and desktop
  • ✅ Customizable appearance and position
  • ✅ Easy to embed on any website
  • ✅ TypeScript support
  • ✅ Modern React with hooks

How to embed on your website:

Method 1: Using script tag with data attributes

<script 
  src="https://anishpradhan.github.io/chat-widget/live-chat-widget.umd.cjs"
  data-api-url="https://your-api.com"
  data-socket-url="wss://your-api.com/ws/chat"
  data-tenant-url="https://your-api.com"
  data-position="bottom-right"
  data-theme="blue"
  data-title="Chat with us!"
  data-subtitle="We typically reply in few minutes."
  data-auto-open="false"
  data-widget="true"
></script>

Method 2: Using JavaScript initialization

<script src="https://anishpradhan.github.io/chat-widget/live-chat-widget.umd.cjs"></script>
<script>
  window.initLiveChatWidget({
    apiUrl: 'https://your-api.com',
    socketUrl: 'wss://your-api.com/ws/chat',
    tenantUrl: 'https://your-api.com',
    position: 'bottom-right',
    theme: 'blue',
    title: 'Chat with us!',
    subtitle: 'We typically reply in few minutes.',
    autoOpen: false
  });
</script>

Method 3: Using ES modules

import LiveChatWidget from 'https://anishpradhan.github.io/chat-widget/live-chat-widget.js';

const widget = new LiveChatWidget({
  apiUrl: 'https://your-api.com',
  socketUrl: 'wss://your-api.com/ws/chat',
  tenantUrl: 'https://your-api.com',
  position: 'bottom-right',
  theme: 'blue',
  title: 'Chat with us!',
  subtitle: 'We typically reply in few minutes.',
  autoOpen: false
});

Configuration Options:

  • apiUrl: Your backend API URL
  • socketUrl: WebSocket URL for real-time chat
  • tenantUrl: Tenant-specific API URL
  • position: Widget position (bottom-right, bottom-left, top-right, top-left)
  • theme: Color theme (blue, green, purple, orange)
  • title: Chat widget title
  • subtitle: Chat widget subtitle
  • autoOpen: Whether to open chat automatically (true/false)
